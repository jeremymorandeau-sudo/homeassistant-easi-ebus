# Icon Placeholder

Pour ajouter une icône à votre addon, placez un fichier `icon.png` ici.

Spécifications recommandées :
- Format : PNG
- Dimensions : 256x256 pixels
- Fond transparent ou blanc
- Style : simple et reconnaissable

Vous pouvez créer une icône qui représente :
- Le logo easi>
- Un symbole de chauffage
- Un bus eBUS
- Une combinaison des éléments ci-dessus
